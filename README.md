# Cott-ADNet
This repository is for the paper: [Cott-ADNet: Lightweight Real-Time Cotton Boll and Flower Detection Under Field Conditions](https://arxiv.org/abs/2509.12442), including the pretrain model and dataset for reproducing.

The paper is accepted by Accepted to HARVEST-Vision 2026 (Second International Workshop on Applications of CV and HPC in Agriculture), in conjunction with IEEE/CVF WACV 2026.
